Styling

Colors

The following CSS Custom Properties are available for styling.

--slider-primary-color
Sets the filled background color of the slider track.
Defaults to #000
--slider-thumb-color
Sets the fill color of the slider thumb
Defaults to #949494
--slider-unfilled
Sets the unfilled background color of the slider track.
Defaults to transparent
--slider-border-color
Sets the border color for the slider track and the thumb.
Defaults to the value of --slider-primary-color

Parts

The following ::parts are exposed:

label
Targets the label element.
input
Targets both input elements.
number
Targets only the input[type="number"] element.
range
Targets only the input[type="range"] element.

Layout

In addition to the default stacked layout, a horizontal/in-line layout is available. It can be used by:

  • Using the in-line boolean attribute on the element.
  • If you're using SCSS, you can use @mixin inline-structure; to recreate the layout.

Events

The slider emits an update custom event.

Contents of the event.detail property of the update event.

slider
A reference to the slider.
value
The value of the slider as of the event.

The body on this page has a update listener attached that will log to the console every time a slider changes.
